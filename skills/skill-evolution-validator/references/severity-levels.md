# Severity Levels

- `P0`: privacy exposure, destructive behavior, automatic high-impact action, or invalid release boundary.
- `P1`: broken Skill structure, failed required behavior regression, unreadable state, or invalid routing ownership.
- `P2`: stale official-source review, incomplete local ledger mapping, or unlabeled evidence that limits confidence.
- `INFO`: expected first-run state or a non-blocking observation.
