# Devolution Ledger

This is a public template for local rule-maintenance notes. Keep real entries local and sanitized. Do not publish private project names, account details, paths, logs, screenshots, transcripts, or credentials.

The updater treats this file as protected personal data. If your local copy changes, future upstream versions are staged for review instead of overwriting it.

## Entry Template

```markdown
## YYYY-MM-DD - short title

- Rule or file:
- Previous lifecycle state:
- Signal:
- Impact:
- Decision:
- Validation:
- Next review:
```
