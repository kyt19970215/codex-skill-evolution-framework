# Local Evolution Change Log

This file is a local-data template. Framework updates never overwrite an installed copy.

Record only durable skill-system changes. Do not paste full prompts, private paths, account data, project secrets, or personal conversation history.

## Entry Template

```text
## YYYY-MM-DD - Short summary
- Scope: global | project | skill
- Action: add | update | relax | merge | archive | remove
- Files:
  - example-skill/SKILL.md
- Evidence: sanitized local locator or recovery note
- Verification: command or behavior check
```
